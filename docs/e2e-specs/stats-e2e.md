# 管理ダッシュボード E2Eテスト仕様書

生成日: 2026-02-06
対象ページ: `/admin/stats`
権限レベル: admin

---

## ユーザーゴール

1. システム全体の利用状況を一目で把握する
2. 質問数の推移を確認する
3. 回答精度（肯定的評価率・類似度スコア）を確認する
4. 回答失敗の頻出キーワードを確認する

---

## テスト環境

```yaml
URL: http://localhost:3247/admin/stats
認証: 必須（ProtectedRoute + admin権限）
テストアカウント:
  email: admin@example.com
  password: admin123
```

---

## E2Eテスト項目一覧（7項目）

| ID | テスト項目 | フロー内容 | 期待結果 |
|----|----------|-----------|---------|
| E2E-STATS-001 | ページ表示・サマリーカード確認 | 管理者ログイン → /admin/stats → サマリーカード表示 | 今月の質問数、肯定的評価率、回答失敗率、登録ドキュメント数が表示される |
| E2E-STATS-002 | 肯定的評価率の詳細確認 | サマリーカード → 肯定的評価率のサブタイトル確認 | 👍 数 / 👎 数が表示される |
| E2E-STATS-003 | 回答失敗率のステータス確認 | 回答失敗率カード確認 | 10%以下ならオレンジ、10%超ならエラー色、サブタイトルに「X件が回答不能」と表示 |
| E2E-STATS-004 | 質問数推移グラフ表示 | グラフエリア確認 | 過去14日間の日別質問数が棒グラフで表示される（データがなければ「データがありません」） |
| E2E-STATS-005 | 不足トピック候補の表示 | 不足トピック候補エリア確認 | 回答失敗質問の頻出キーワードが最大8件チップ表示される（なければ「回答失敗がありません 🎉」） |
| E2E-STATS-006 | 平均類似度スコアの確認 | 検索品質指標エリア確認 | 平均類似度スコアが%表示され、70%以上=良好、50-70%=要注意、50%未満=要改善のチップが表示される |
| E2E-STATS-007 | データなし状態の表示 | 質問履歴が空の状態でページ表示 | 全カードに0が表示され、グラフ・不足トピックは「データがありません」と表示される |

---

## 備考

- グラフの最大高さは150pxで、最大値に対する割合で棒が描画される
- 不足トピック候補は回答失敗質問（is_no_answer=true）から頻出キーワードを抽出
- 類似度スコアは0.0〜1.0の値で、70%以上が良好とされる
